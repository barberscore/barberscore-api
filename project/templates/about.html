About Barberscore

My name is David Binetti and I've sung with the Alexandria Harmonizers and Voices in Harmony. I built this app because I wanted a mobile-friendly website that I could use to help track the convention performances.

I will not be using or sharing your personal information in any way. I'm just a barbershopper/geek and thought it would be a cool app.

Source code for this app can be found here. If you have any questions or need support open an issue there.
